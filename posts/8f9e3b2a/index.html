分子动力学后处理（1）——定向四面体有序度（OTO）参数计算

发表于2026-07-22|更新于2026-07-22
|总字数:2.8k|阅读时长:10分钟|浏览量:

引言

水是自然界中最常见却又最奇特的液体之一。水的许多反常性质——如密度在4°C时最大、高比热容、随压力增加黏度先降低后升高等——都源于其独特的氢键网络结构。在分子动力学模拟中，如何定量描述水分子局部结构的"有序程度"一直是一个核心问题。

定向四面体有序度（Orientational Tetrahedral Order，OTO）正是为此而生的一个参数。它衡量的是：给定一个水分子，它的四个最近邻水分子在空间上是否接近一个理想的正四面体构型。这个参数最早由 Chau 和 Hardwick（1998）提出，现已成为水溶液模拟中最常用的局部结构分析工具之一。

在这篇教程中，我将：

  1. 解释 OTO 参数的数学定义和物理意义
  2. 基于 GROMACS 模拟轨迹，使用 MDAnalysis 逐行编写计算脚本
  3. 分析计算结果并讨论其意义

1. 定向四面体有序度的数学定义

定向四面体有序度参数 q 的定义如下：

q=1−38∑i=13∑j=i+14(13+cos⁡θijk)2q = 1 - \frac{3}{8} \sum_{i=1}^{3} \sum_{j=i+1}^{4} \left(\frac{1}{3} + \cos \theta_{ijk}\right)^2 q=1−83​i=1∑3​j=i+1∑4​(31​+cosθijk​)2

其中：

  • 对每个目标水分子，找到其四个最近邻水分子（以氧原子 OW 位置为准）

  • \theta_{ijk}$$ 是由参考水分子氧原子（顶点）与其两个相邻水分子氧原子组成的夹角，即 $$\angle O_j O_{ref} O_k

  • 对所有可能的 4 个近邻氧原子配对（共 C(4,2)=6 个夹角）求和

物理意义：

  • q = 1：完美的正四面体构型，对应理想冰 Ih 中的水分子

  • q ≈ 0：完全无序的随机分布

  • 对于体相水（液态水，~300K），平均 q ≈ 0.5 - 0.6

为什么是四个近邻？ 因为每个水分子最多可以形成四个氢键（两个供体、两个受体），因此最近邻的四个水分子的空间排布直接反映了氢键网络的局域完整性。

2. 计算流程概览

整个计算流程如下：

读取 GROMACS 轨迹（.tpr + .xtc） ↓ 选定目标残基（如残基 15 周围的 5 Å 以内） ↓ 选定更宽的溶剂缓冲区域（8.5 Å） ↓ 遍历每一帧 ├── 对每个水分子： │ ├── 找出该帧内与之距离最近的 4 个水分子 │ └── 计算 q 值 └── 收集所有 q 值 ↓ 绘制 q 值概率分布图 ↓ 保存 q 值到文本文件

3. 代码逐行解析

3.1 引入依赖

python import MDAnalysis as mda import numpy as np import matplotlib.pyplot as plt

  • MDAnalysis：用于读取和操作 MD 轨迹。支持 GROMACS、AMBER、LAMMPS、CHARMM 等多种格式。
  • NumPy：数值计算基础库。
  • Matplotlib：数据可视化。

3.2 分布统计函数

这个函数将连续的 q 值分布离散化为直方图：

  • 将区间 [0, 1] 等分为 bin 个窗口
  • 统计每个窗口内的数据点个数
  • 转换为百分比概率分布
  • 返回每个窗口中心位置的 x 值和对应的概率

注意：虽然理论上 q 可以接近甚至略小于 0，但 0-1 的统计区间对液水已足够覆盖绝大多数情况。

3.3 查找最近四个的近邻

将所有水分子按距离排序，返回距离最近的四个水分子的索引。这是核心步骤——找到每个目标水分子的四个最近邻。

3.4 计算三点夹角余弦

计算 $$\angle ABC$$（顶点 B）的余弦值。在水的 OTO 场景中，A 和 C 是两个近邻水的氧原子坐标，B 是参考水的氧原子坐标，即计算 $$\cos \angle O_j O_{ref} O_k$$。

3.5 计算单个水分子的 q 值

这是公式的直接实现：

  • 初始化 q = 1（假设完美四面体）
  • 对四个近邻的 6 个配对（i, j）逐一计算
  • 每个配对减去$$ (3/8)(1/3 + \cos\theta)^2$$
  • 最终 q 越接近 1，局域结构越接近正四面体

3.6 计算一帧中所有水分子的 q 值

对于一帧中的每个水分子：

  1. 计算它与所有其他水分子的距离
  2. 找出最近的四个水分子
  3. 计算这四个水分子的四面体有序度

这里使用了两个选择：

  • waters：目标残基外围 5 Å 以内的水分子（目标水分子）
  • water_buffer：目标残基外围 8.5 Å 以内的水分子（候选近邻集合）

这样设计的目的是：既要确保有足够的候选近邻，又要避免包含过多远处的水分子拖累计算。

3.7 主程序

关键点说明：

参数 含义
md.tpr GROMACS 拓扑/参数/输入文件
md_skip.xtc 压缩轨迹文件
esid 15 目标残基编号（在 MDAnalysis 中索引从 0 开始，实际 GROMACS 的 16 号残基对应 resid 15）
round 5 选择目标残基周围 5 Å 以内的水分子
updating=True 随着轨迹载入更新选择（中心分子可能移动）

esid 15。

4. 运行结果与讨论

4.1 结果可视化

运行脚本后生成的分布图如下：

从直方图可以看到：

  • q 值的分布范围大致在 -0.3 到 1.0 之间
  • 主峰集中在 q ≈ 0.5 - 0.7 之间

4.2 结果分析

esult.txt 中记录了所有计算得到的 q 值，部分示例如下：

0.889 — 高有序度，接近理想四面体 0.679 — 中等有序度 0.147 — 低有序度，结构被破坏 -0.251 — 极低有序度，可能处于界面或剧烈扰动区域 0.987 — 非常接近完美的四面体

一些关键观察：

q 值范围 物理含义
高值（q > 0.8） 水分子处于类冰环境的四面体配位中，常见于远离溶质的体相水或疏水空腔内
中等值（q ≈ 0.5-0.7） 典型的液态水四面体有序度，反映氢键网络的动态波动
低值（q < 0.3） 局部四面体结构严重扭曲，常见于溶质第一水合壳层或高扰动区域
负值（q < 0） 极端四面体畸变，四个最近邻几乎共面或非常不对称

4.3 从结果可以解读什么？

  • 水合层结构：通过比较不同位置（如体相 vs. 界面 vs. 配位层）水的 q 分布，可以定量描述溶质对溶剂结构的扰动程度
  • 疏水效应：疏水溶质周围的水分子通常 q 值升高（更有序），形成所谓的"笼形"结构
  • 温度效应：温度升高时，q 分布整体左移，峰值降低、展宽增大
  • 压力效应：高压下 q 分布变宽，反映四面体结构受压变形

将脚本 OTO.py 与以下 GROMACS 文件放在同一目录：
- md.tpr — GROMACS 拓扑和参数文件
- md_skip.xtc — 压缩的轨迹文件（可跳过部分帧以减小计算量）

然后执行：

脚本会弹出 Matplotlib 窗口显示分布图，同时生成

### 5.3 性能优化建议

当前实现是逐帧逐水分子全遍历，随着体系增大计算成本会显著增加：

| 优化策略 | 说明 |
|---------|------|
| **使用 KDTree** | scipy.spatial.cKDTree 可将近邻搜索从 O(n²) 降至 O(n log n) |
| **并行化** | 使用 multiprocessing 对帧级循环并行 |
| **降采样** | 跳过部分帧（如改为 	rajectory[::10]）在保证统计意义的同时大幅减少计算量 |
| **使用 MDAnalysis 内置方法** | mda.lib.distances.self_distance_array 比逐对计算更高效 |

## 6. 扩展：二阶定向有序度参数

OTO 参数主要描述**第一水合壳层**的四面体特征。如果需要分析**更长程的序结构**，可以扩展为二阶定向有序度：

$$
q^{(2)} = 1 - \frac{3}{8} \sum_{i=1}^{4} \sum_{j=i+1}^{4} \left(\frac{1}{3} + \cos \theta_{ijk}^{(2)}\right)^2
$$
其中$$\theta_{ijk}^{(2)}$$由目标水分子的四个**第二近邻**水分子构成。这种方法可以对水化壳层进行"层析"分析。

本文详细介绍了定向四面体有序度（OTO）参数的计算方法：
- **理论**：回顾了 q 参数的数学定义和物理意义
- **代码**：逐行解读了基于 MDAnalysis 的实现
- **应用**：讨论了如何从计算结果中提取物化信息

OTO 参数是理解水在各种化学环境和生物环境中行为的强大工具——从蛋白质折叠到离子输运，从冰成核到界面催化。掌握这个简单的参数计算，是深入分析水溶液模拟的第一步。
